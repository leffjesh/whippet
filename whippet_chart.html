Clackamas River Basin Weed Observation Prioritization Dataset Analysis

Data compiled using iMapInvasives Oregon dataset and prioritized using a modified WHIPPET algorithm. Make this better by sharing your data if you haven't already. Check out the Oregon iMapInvasives resources page for more information. Analysis and charts developed by Jeff Lesh, jlesh@conservationdistrict.org. Feel free to send me questions if you have any.
